以一个字作为起点来发散思维，联想一些与这个字相关的人与事与物，这个想法很早以前就有，来源于之前片刻上的一个模块，叫词卡，之前也断断续续写过一些，这次决心重启这个计划。

重启后的第一个字，选择了梅，来自这个时节最值得看的花-梅花。

梅兰芳——美的创造者

说到梅，除了那几首写梅的诗词，大家第一个想到的应该是梅兰芳大师吧。梅兰芳先生是京剧的大师，是毫无疑问的，对于京剧我不懂，所以我不多说。按照现在的思路，去评价和了解一个人，总离不开这个人的感情史，所以我去了解了一下梅先生的感情史。

梅先生有三个妻子，而且，可以说三个都是正妻。

17岁那年，梅先生遇到了他的第一位妻子——王明华，开启了他们今后十几年的羁绊。王明华是个小女人，她满足于照顾丈夫与子女公婆，只要儿女双全、夫妻和美、丈夫事业有成，她就可以很满足。然而有时候她又像男子一样果决，在有了一对子女之后，她毅然决然地做了绝育手术，只为了不让自己的丈夫牵挂。

然而，王明华的果断，像飞蛾扑火一样，把自己烧了个遍体鳞伤，在儿女因为荨麻疹而去世之后，儿女的去世让她痛苦不堪，绝育的决定更是将她推向了更深的深渊。

“不孝有三无后为大”，那个时代，女人最大的作用之一，大概就是生儿育女了，不能为梅家传宗接代，无论她怎样能干贤惠，任劳任怨，终究引人非议，无论是梅兰芳还是梅家其他人都不能接受这样一个结果：梅家要无后了！尤其是在梅兰芳兼祧两房的情况下。这里说一下兼祧两房的意思，兼祧就是指兄弟数人，其中一个没有儿子的，便让另一个兄弟的儿子继嗣两房，俗称一子顶两门，兼祧两房可分别为其娶两房妻子，以传两门的后代。梅兰芳先生因为父亲早逝，一直被记在无子的大伯名下，自然而然就有“兼祧两房”的责任。

在这种情况下，传宗接代的压力，让王明华不得不接受丈夫再娶一个妻子，就是梅先生的第二个妻子，也是陪伴梅先生最久的一位——福芝芳。

福芝芳的入门本就是为了解决无后的问题，所以梅先生也说不上对其有什么感情，只能说是恰逢其会，师出同门，又有“梅党”的老干部冯六爷一语中的，指出福芝芳“长得不错，唱得不错，能生孩子”，正中梅家下怀。然而福芝芳此人不简单，在某种程度上，她做的比王明华更好，无论是事业还是家庭，甚至在家庭关系上。

然而越是这样的女人越不能接受自己的丈夫有别的女人，王明华是因为香火问题，不得不接受福芝芳，福芝芳则因为后入门的原因，尊敬王明华，但是这两个人都不能接受丈夫区第三位妻子，孟小冬就是这个第三人。

被冠以“冬皇”之名的孟小冬，在18岁的时候遇到了梅先生，那个年纪的女生总是天真想要嫁一个大英雄，在孟小冬眼里，梅兰芳先生无疑是个大才子，大英雄。郎有情，妾有意，同台演出了几场，两人情愫暗生。

又是那个冯六爷出来搞事情，当时的王明华重病在身，在天津养病，梅家内务基本就全权在福芝芳手上了，梅党的人怎能允许一家独大，于是冯六爷，齐如山等梅党人士又开始搞东搞西，撮合了孟小冬和梅先生，这一场不那么美好的婚姻就这样开始了。

从一开始就带有欺骗隐瞒性质的感情大抵都不会有好结果。由于王明华长期重病在床，梅先生兼祧两房，所以在正式成婚前，给孟小冬的名分是和福芝芳一样的正妻身份，然而嫁是嫁了，却嫁的那么委屈，那么残忍，嫁给梅兰芳的孟小冬被拦在了梅家门外，福芝芳死活的不让把孟小冬挡在了门外。年轻的女孩对爱情的期望仍然大的让她看不清现实，爱情的美好让她忽视了爱情的背后那么多的坎坷和荆棘，孟小冬接受了金屋藏娇，暂居缀玉轩，深入简出。

悲剧的种子埋下，之后的“冯宅枪击案”成了催化剂。

婚后不久，一位迷恋孟小冬的粉丝（一说是学生李志刚，一说是市长公子王惟琛）得知梅孟结婚之后，怀恨在心，持枪闯入缀玉轩，直言要见梅兰芳，慌乱之中错杀了当时正好在场，前来出面交涉的梅兰芳好友《大陆晚报》经理张汉举。

这种事放在现代那也是满城风雨呀，更何况当时，更是涉及到了梅兰芳先生这位大明星，一时间舆论铺天盖地。这对于梅兰芳和梅家来说，影响太大了，为了避讳，梅兰芳开始逐渐疏远孟小冬，也不再去孟小冬那里了，只想等风头过去再说。

这之后梅先生与孟小冬的感情每况愈下，戴孝风波，使梅兰芳与孟小冬终于走到了无可挽回的地步。

1930年，由于兼祧两房，等同于梅先生母亲的伯母去世。

孟小冬听闻也照规矩剪了短发，头插白花，披麻戴孝，以儿媳的身份前去守灵，此时孟小冬嫁给梅先生已近四年，却是第一次进入梅家门，然而这一次依然失败了。

福芝芳叫人拦在门口，用孩子威胁梅兰芳，不让孟小冬进门，最后，梅兰芳只好说你回去吧，你回去。

冬皇孟小冬能在舞台上够征服所有人，却无法为自己博得一个应有的名分。

梅先生的一句‘你回去吧’，打破了孟小冬最后的幻想与希望，梅兰芳终究不是她的大英雄，保护不了她。

之后孟小冬就在《大公报》登启事，单方面宣布决裂，如此之决绝。

“冬当时年岁幼稚，世故不熟，一切皆听介绍人主持。名定兼祧，尽人皆知。
乃兰芳含糊其事，于其母去世之日，不能实践前言，致名分顿失保障，毅然与
兰芳脱离家庭关系。是我负人？抑或负我？世间自有公论，不待冬之赘言。”

四年多的感情，就这样落幕，而那时，孟小冬依旧没有得到当时说好的正妻的名分。

梅先生一生的三个女人，王明华宽容，福芝芳智慧，孟小冬清高。

梅妻鹤子——林和靖

说起菊，我们想到的是陶渊明，说起莲，我们想到的是周敦颐，而说起梅，我们就会想到林和靖了。

林和靖，也就是林逋，字君复，被称为和靖先生。

同陶渊明一样，和靖先生也是以隐士出名的（吐槽的说，隐士还这么出名，真是有点矛盾），和靖先生后来隐居西湖孤山，终生不仕不娶，却放不下两个爱好，一是种梅，二是养鹤，自谓 “以梅为妻，以鹤为子”，人称“梅妻鹤子”。

林和靖爱梅，相传，他绕屋种梅三百株，将每一株梅子卖得的钱，包成一包，投于瓦罐，每天随取一包作为生活费，待瓦罐空了，刚好一年，新梅子又可兑钱了。这和唐寅的《桃花庵歌》似乎有点相像。

桃花坞里桃花庵，桃花庵下桃花仙。
桃花仙人种桃树，又摘桃花换酒钱。

不过和靖先生比唐伯虎大概幸福得多了，他少年游学，足迹遍于江淮之间，从四十多岁后隐居杭州孤山，直到八十三岁死于孤山，葬于孤山，在孤山隐居期间，他读书、种梅、写诗。

众芳摇落独暄妍，占尽风情向小园。
疏影横斜水清浅，暗香浮动月黄昏。
霜禽欲下先偷眼，粉蝶如知合断魂。
幸有微吟可相狎，不须檀板共金尊。

林和靖是一位隐者，大概到了中年，他把红尘的种种都看得明明白白了，他与世隔绝，隐匿深山，逍遥方外，不问红尘。

有人考证说，林和靖并非不娶，而是丧偶后不再续娶，自别家人，过着“梅妻鹤子”的隐居生活。

张岱在《西湖梦寻》也说，南宋灭亡后，有盗墓贼挖开林逋的坟墓陪葬的竟然只有一只端砚和一支玉簪，端砚若是和靖先生自用之物，那只玉簪大概是和靖先生亡妻的信物吧。

我们这样推断，和靖先生少年游学中邂逅了一个心爱的姑娘，他与她共同游历江淮，赏梅，赏鹤，然而，这位姑娘离开了人世，和靖先生大悲之下，看透了红尘，隐居孤山，梅妻鹤子，只留下玉簪和这阙词来怀念故人。

吴山青，越山青。两岸青山相送迎，谁知离别情？
君泪盈，妾泪盈。罗带同心结未成，江头潮已平
